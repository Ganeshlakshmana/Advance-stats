# Advanced Statistics — Ganesh Lakshmana

A Jupyter Notebook containing advanced statistical analysis and visualizations, covering key topics in probability and statistics.

## 📘 Overview

This notebook implements and visualizes several fundamental statistical concepts using Python. Each task explores a different distribution or statistical method with clear outputs including plots, tables, and computed statistics.

## 🗂️ Contents

| Task | Topic |
|------|-------|
| **Task 0** | Discrete Distributions (Bernoulli, Poisson, Negative Binomial, Geometric) — PMF/CDF, Mean, Median, Mode |
| **Task 1** | Negative Binomial Distribution — Meteorite count modeling |
| **Task 2** | Continuous Survival/Waiting Time Analysis — PDF, Survival Function, Quartiles |
| **Task 3** | Further Statistical Analysis |
| **Task 4** | Further Statistical Analysis |
| **Task 5** | Hypothesis Testing |
| **Task 6** | Bayesian Analysis |

## 🛠️ Dependencies

- Python 3
- [NumPy](https://numpy.org/)
- [SciPy](https://scipy.org/)
- [Matplotlib](https://matplotlib.org/)
- [Pandas](https://pandas.pydata.org/)

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Ganeshlakshmana/Advance-stats.git
   ```

2. Install the required libraries:
   ```bash
   pip install numpy scipy matplotlib pandas
   ```

3. Open the notebook:
   ```bash
   jupyter notebook Advanced_statistics_Ganesh_Lakshmana.ipynb
   ```
